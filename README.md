# 🚀 ZRS Carpet - Server Side

The backend server-side application for the **Carpet Selling Zrs** platform. Built with Node.js, Express.js, and MongoDB.

## 🛠️ Tech Stack
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Environment:** `dotenv`
